Methodology

All languages implement the same ping-pong benchmark: two actors exchange 10 million messages (20M total round-trips). Each language uses its native, idiomatic concurrency primitives with no artificial handicaps.

  • Throughput: Messages processed per second (higher is better)
  • Latency: CPU cycles per message at ~3GHz (lower is better)
  • Memory: Resident Set Size during execution (lower is better)
